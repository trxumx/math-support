# МО СППР — подготовка к сессии

Учебное SPA по курсу **Математическое обеспечение СППР**: 38 вопросов с эталонными ответами + конспект-методичка по безусловной оптимизации.

🔗 **Открыть:** см. GitHub Pages этого репозитория.

## Возможности

- **📖 Чтение** — все вопросы с поиском, оглавлением и отметками «знаю / повторить / не знаю».
- **🃏 Карточки** — флешкарты с перемешиванием, фильтром по слабым местам и горячими клавишами.
- **🎯 Экзамен** — случайные N вопросов с самооценкой и сводкой «что повторить».
- **📚 Теория** — конспект-методичка с навигацией по разделам.
- Прогресс сохраняется в `localStorage`.
- Тёмная и светлая темы.
- KaTeX для математики, рендер на клиенте.

## Горячие клавиши

| Где | Клавиша | Действие |
|---|---|---|
| Карточки | `←` / `→` | предыдущая / следующая |
| Карточки | `Space` | перевернуть |
| Карточки (после флипа) | `1` / `2` / `3` | не знаю / повторить / знаю |
| Экзамен | `Space` | показать ответ |
| Экзамен | `1` / `2` / `3` | не ответил / частично / хорошо |

## Локальная разработка

```bash
python3 build.py          # перегенерить data.js из .md
python3 -m http.server 8765
# открыть http://localhost:8765
```

## Структура

```
build.py     # парсит два .md в data.js
index.html   # SPA
styles.css   # тёмная/светлая темы
app.js       # вся логика
data.js      # сгенерированные вопросы и теория
Ответы_на_вопросы_сессия_МО_СППР.md   # исходник вопросов
Конспект_методичка_безусловная_оптимизация.md   # исходник теории
```
